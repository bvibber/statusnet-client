This is StatusNet Desktop 0.9a <http://status.net/desktop>, a
cross-platform StatusNet client developed using Appcelerator
Titanium Desktop 1.2.1. It runs on Linux, Mac OS X, and Windows
(Win2K/XP/Win7). StatusNet Desktop is Copyright 2010, StatusNet, Inc.
<http://status.net/> and licensed under the Apache License, Version 2.0
<http://www.apache.org/licenses/LICENSE-2.0>.

Partial Feature list:

- Multi-platform: runs on Linux, Mac OS X, and Windows (2000/XP/Vista/Win7)
- Connects with multiple StatusNet sites, including secure and private installations
- Auto-refresh of visible timeline
- Notifications for new notices
- View public, user, friends, replies, tag and group timelines
- Subscribe/unsubscribe other users
- Join/Leave groups
- View inbox and send direct messages
- Post new notices and @-replies
- Notice search
- Notice caching

Bug tracker: http://status.net/open-source/issues

Building
--------

The source code for StatusNet Desktop is located on Gitorious
http://gitorious.org/statusnet-client

To build the app you'll need Titanium Developer 1.2.1
http://www.appcelerator.com/products/download/

Once you get Titanium Developer installed you should be able to import
the project using the "Import project" button and build it by hitting
the "Launch" button from the "Test and Package" tab.  See the Titanium
documentation for more details:

http://developer.appcelerator.com/documentation
